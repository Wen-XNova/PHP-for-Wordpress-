# PHP-for-Wordpress-
The code for a WP portfolio project, using an OCI node via a manual LEMP stack.

**[ LIVE_DEMO: https://wen-dev.duckdns.org]**

Infrastructure: Deployed on an Oracle Cloud Infrastructure (OCI) Compute Instance. Featuring a manually configured Linux environment and a secure SFTP deployment workflow, bypassing automated "one-click" installers.

Advanced CSSOM Manipulation: Implemented a non-linear layout engine using CSS Grid and the order property. This allows the UI to dynamically reorder the DOM visually—promoting active project nodes to the primary position—without breaking the document flow or requiring heavy JS re-renders.
